#  OLA Rides Analysis — Bengaluru
##  Project Overview
End-to-end data analysis project on 50,000+ OLA ride 
records from Bengaluru city. This project analyzes 
booking patterns, cancellation reasons, revenue trends, 
and ride performance using SQL, Python, and Power BI.

##  Business Problem
OLA wants to understand:
- Why are 33% of rides getting cancelled?
- Which vehicle types perform best?
- What are peak demand hours?
- How is revenue distributed across vehicle types?

##  Tools Used
| Tool | Purpose |
|------|---------|
| MySQL | Data cleaning & analysis |
| Python (pandas, matplotlib, seaborn) | EDA & visualizations |
| Power BI | Interactive dashboard |

##  Dataset
- Source: GitHub (OLA DataAnalyst Project)
- Records: 50,000 rides
- Period: 2024
- City: Bengaluru, India
- Columns: 21 features

##  Key Insights
1. 67% rides successful — business is healthy
2. Driver cancellations at 19.2% — biggest challenge
3. Bike is most booked vehicle type
4. Total revenue: ₹34.27M across all vehicle types
5. All payment methods equally preferred (~25% each)
6. Ride demand consistent 24/7 — no single peak hour
7. "More than permitted people" = #1 driver cancel reason
